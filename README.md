# petroleum-generation

Модуль для моделирования процессов нефтегазогенерации и термического созревания органического вещества.

## Описание

Этот Python-пакет содержит реализации различных моделей для:
- Однокомпонентной и многокомпонентной генерации углеводородов
- Моделей отражательной способности витринита (термического созревания)
- Трансформации иллита
- Преобразования биомаркеров
